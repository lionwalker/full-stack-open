# Full Stack Open 2024

This repository contains my exercise submissions for the [Full Stack Open](https://fullstackopen.com/en/) course offered by the University of Helsinki.

## About the Course

Full Stack Open is a comprehensive introduction to modern web application development with JavaScript. The course focuses on building single-page applications with React and supporting them with RESTful and GraphQL web services implemented with Node.js. It also covers TypeScript, React Native, and Continuous Integration/Continuous Deployment (CI/CD).

## Course Contents

- **Part 0:** Fundamentals of Web Apps
- **Part 1:** Introduction to React
- **Part 2:** Communicating with Server
- **Part 3:** Programming a Server with Node.js and Express
- **Part 4:** Testing Express Servers, User Administration
- **Part 5:** Testing React Apps
- **Part 6:** Advanced State Management
- **Part 7:** React Router, Custom Hooks, Styling with CSS and Webpack
- **Part 8:** GraphQL
- **Part 9:** TypeScript
- **Part 10:** React Native
- **Part 11:** CI/CD
- **Part 12:** Containers
- **Part 13:** Using Relational Databases
